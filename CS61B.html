Introduction

CS61B is one of the most significant computer science courses in US Berkerley. It aims for teaching some basic knowledge about Java and some of the most vital data sturctures that you might be used in the future. There are lots of interesting labs and projects in this course.


There are two free versions of courses that can be used to learn, since these two versions have open the autograder to public, with which you can verify your assignments in this course, and the instrutor of these two versions is a very interesting and humourous teacher whose name is Josh Hug. You can see webpages of this course in Spring 2018 or Spring 2021.
